# OULECU v2.8.0

**What's New in v2.8.0:**
- 🎨 **Dynamic Theme Logo**: The OULECU logo inside cards and throughout the app now perfectly tints to match your chosen template accent colors.
- 📐 **15 Custom Corner Styles**: Enjoy beautifully rendered card shapes including Ticket, Scalloped, Cutout, Notched, Zigzag, Hexagon, Octagon, Leaf, Folded, Torn, and Wave, all correctly rendered in both preview and the final image!
- 🎨 **Dynamic Shadow Theming**: The shadow behind cards now dynamically adapts to the selected template's accent color instead of defaulting to gold.
- 📱 **Perfect Native Navigation**: You can now seamlessly navigate back and forth during content creation using Android's native back button without accidentally closing the app.
- 🪟 **Preview Screen Overhaul**: Fixed an issue where fonts would become microscopically small during preview scaling, ensuring previews perfectly match the generated layout.
- 👤 **Clean Handle UI**: Removed the hardcoded repeating "@" symbol when updating your user handle in settings, and defaulted the base accent colors to clean White.
- ⚡ **Fixed "Next" Button Freeze**: Content typing now properly updates the UI state in real-time, preventing the "Next" button from getting stuck after entering content.
